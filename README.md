# Analisis Aksesibilitas Hotel terhadap Halte TransJakarta

Proyek ini menganalisis keterjangkauan hotel terhadap halte TransJakarta menggunakan:
- Buffer 500 meter (analisis spasial)
- K-Means Clustering (unsupervised machine learning)
